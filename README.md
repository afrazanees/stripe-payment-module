# Stripe Payment Module

Stripe payment integration. Node.js backend + vanilla JS frontend.

## Structure

```
backend/
├── server.js
├── routes/
│   ├── payment.js
│   └── webhooks.js
├── middleware/
│   ├── errorHandler.js
│   └── validators.js
└── utils/
    └── logger.js

frontend/
├── index.html
├── payment-form.js
├── styles.css
└── config.js
```
